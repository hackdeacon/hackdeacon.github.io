# Personal Portfolio Website | hackdeacon.github.io

This is the source code for my personal portfolio website, hosted on GitHub Pages. The website showcases my skills, projects, and content channels.

## ✨ Features

*   **Modern & Responsive Design:** A clean, single-page layout with a "black, white, silver-gray, with a touch of pink" theme, fully responsive for both desktop and mobile devices.
*   **Smooth Animations:** Subtle scroll effects and animations to enhance the user experience.
*   **Sections:**
    *   **Home:** A brief introduction.
    *   **Skills:** Highlights my expertise in front-end development, tech, and media creation.
    *   **Projects:** Showcases selected works.
    *   **Channels:** Links to my social media and content platforms.

## 🛠️ Tech Stack

*   **HTML5**
*   **CSS3:** For styling and layout, including flexbox and grid.
*   **JavaScript (ES6+):** For DOM manipulation and interactive features like smooth scrolling and scroll-triggered animations.

## 🚀 Deployment

This site is automatically deployed and hosted via [GitHub Pages](https://pages.github.com/). Any push to the `main` branch will trigger a new deployment.

You can view the live site at: [https://hackdeacon.github.io/](https://hackdeacon.github.io/)
