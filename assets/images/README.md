# 图片资源文件夹

此文件夹用于存放个人主页所需的图片资源。

## 文件夹结构

- `avatar/` - 存放头像图片
  - 建议尺寸：400x400px 或更大（正方形）
  - 建议格式：PNG、JPG、WebP
  
- `logo/` - 存放 Logo 图片
  - 建议尺寸：根据使用场景调整
  - 建议格式：PNG（支持透明背景）、SVG
  
- `covers/` - 存放项目封面图片
  - 建议尺寸：1200x600px 或更大
  - 建议格式：JPG、PNG、WebP

## 使用说明

将对应的图片文件放入相应的文件夹后，更新 `index.html` 中的图片路径引用。
